Project Info


Django

Django To-Do List is a task management web application developed using the Django framework. It allows users to easily create, update, and delete tasks to keep their daily activities organized. The application provides a clean and user-friendly interface where users can track their tasks, mark them as completed, and manage their workflow efficiently. It also includes features such as user authentication, ensuring that each user can securely access and manage their own tasks. With a responsive design, the Django To-Do List application works smoothly across different devices, helping users stay productive and organized anytime and anywhere.


Tea Station

Tea Station is an e-commerce web application developed using the Django framework that allows users to explore and purchase a wide variety of teas online. The platform provides a clean and user-friendly interface where customers can browse different tea products, view details, and add their favorite items to a shopping cart. It also includes user authentication so customers can securely create accounts, log in, and manage their orders. With its responsive design and simple navigation, Tea Station offers a smooth and convenient online tea shopping experience.


SpotStar

SpotStar is a music streaming web application developed using the Django framework that allows users to discover and enjoy a wide range of songs in one place. The platform provides a simple and user-friendly interface where users can browse different music categories, listen to their favorite tracks, and create personalized playlists. It is designed to make music access easy and enjoyable, offering a smooth streaming experience while helping users organize and manage their favorite songs.
